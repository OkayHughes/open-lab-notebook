We define (u⊗v)w≡(w⋅v)u (\boldsymbol{u} \otimes \boldsymbol{v}) \boldsymbol{w} \equiv (\boldsymbol{w} \cdot \boldsymbol{v}) \boldsymbol{u} (u⊗v)w≡(w⋅v)u to be the dyadic product (specific case of tensor product). Kinematics

We assume we have a map ϕ:R3×R+→R3\phi : \mathbb{R}^3 \times \mathbb{R}^+ \to \mathbb{R}^3ϕ:R3×R+→R3 which maps X→ϕ(⋅,t)x \boldsymbol{X} \stackrel{\phi(\cdot, t)}{\to} \boldsymbol{x}X→ϕ(⋅,t)x

The lagrangian ("material") description of density and the eulerian ("spatial") description can therefore be unified as ρ=ρ(X,t)=ρ(ϕ(X,t),t) \rho = \rho(\boldsymbol{X}, t) = \rho(\phi(\boldsymbol{X}, t), t)ρ=ρ(X,t)=ρ(ϕ(X,t),t), showing that ϕ\phiϕ is identified with the usual "flow map".
