Linear structure

Why would we want more generality than this?

"Linearity" typically looks like various extensions of the 1D linear system y=mx+by=mx+by=mx+b. We want to be able to multiply by coefficients (e.g., mmm) and add potential solutions (x,bx, bx,b) together. You can verify using the laws of matrix multiplication that A(x+y)=Ax+AyA(\mathbf{x} + \mathbf{y}) = A\mathbf{x} + A\mathbf{y}A(x+y)=Ax+Ay, and that A(ax+y)=aAx+AyA(a\mathbf{x} + \mathbf{y}) = aA\mathbf{x} + A\mathbf{y} A(ax+y)=aAx+Ay (a∈Ra \in \mathbb{R}a∈R). The adjoint, and bilinear mappings

While the adjoint of an operator can be defined without any reference to an inner product, it is much easier to talk about them when we do have an inner product.
